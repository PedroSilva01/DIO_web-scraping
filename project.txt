1- O QUE É WEB SCRAPING?
    WEB SCRAPING É UMA TÉCNICA PARA COLETAR DADOS DE SITES, QUANDO VOCE ENCONTRA DADOS NAQUELE SITE, MAS LE NÃO DISPONIBILIZA UMA API.
    ELA PODE SER CONSIDERADA UMA TÉCNICA MALICIOSA, POIS NÃO É QUALQUER SITE QUE VOCÊ PODE IR RASPANDO OS DADOS, POR CONTA DOS DIREITOS AUTORAIS E ETC, POIS PODE QUEBRAR A LGPD E ACAVAR SENDO PROCESSADO.
    UTILIZAREMOS A LIB BeautifulSoup.
    IREMOS COLETAR DADOS DO BLOG DA DIO.
